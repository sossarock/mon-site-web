# AgriConnect-Bj

**Plateforme e-commerce pour les produits agricoles du Bénin**

AgriConnect-Bj connecte directement les agriculteurs et les acheteurs. 
L'objectif est de vendre des produits frais : maïs, igname, riz, haricot... sans intermédiaire.

## 🎯 Objectifs du projet
- Faciliter l'achat de produits agricoles locaux au Bénin
- Donner de la visibilité aux agriculteurs
- Avoir un site rapide et accessible sur téléphone

## 👥 Utilisateurs cibles / Personas
1.  **Acheteur particulier** : Habitant de Cotonou/Abomey-Calavi qui veut des produits frais
2.  **Agriculteur/Revendeur** : Qui veut vendre sa production en ligne

## 🛠️ Technologies utilisées
- **HTML5** : Structure sémantique
- **CSS3** : Design responsive pour mobile
- **GitHub Pages** : Hébergement gratuit
